    <div style="
      line-height: 1.8;
      text-indent: 2em;
    ">
      <p style="margin: 0 0 1.5rem 0;">《人偶游戏》这本书是我去新加坡前在书店买的，当时还和大潘借了50，最后忘记了从新加坡回来才想起来还钱，还真是最近有点健忘</p>
      <p style="margin: 0 0 1.5rem 0;">我仿佛想起了之前看的江户川乱步的《幽灵塔》，感觉风格很像</p>
      <p style="margin: 0 0 1.5rem 0;">我看到快结尾时候，总有一种想法：估计又是和那些推理小说一样，前中期看着很神秘惊悚，但一揭示结局真相就会让我觉得没啥新鲜的就那回事，看到后半段就能猜出结局</p>
      <p style="margin: 0 0 1.5rem 0;">但是这本书不一样，他到最后也没有完完全全揭示谜底，真相是什么永远的埋藏掉了，许多埋下的伏笔似乎也没有解释，但这样代入感也确实强，在男主的视角，哪怕是最后的结局，那些不可能知道的或许也永远不可能知道了吧</p>
      <p style="margin: 0 0 1.5rem 0;">我感觉比之前看的那本同学推荐的变格推理小说《永劫馆》强，那本书似乎落入了“好像也没什么”的陷阱，看着有点乏味，看了几十本东野圭吾写的推理小说，这本也一如既往地不以解密为主而是揭示人性为主，比起设计一个谜题东野圭吾他更喜欢写一个故事，不同的是其他的故事在最后往往会有个清晰的全貌解释，但是这本最后也没有一个完整真相，这样的留白或许赋予了这本书某些独特的不同于往常作品的独特韵味</p>
